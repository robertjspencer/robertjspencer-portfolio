Robert Spencer Personal Website

This repository contains the source for a personal website branded for
Robert Spencer and intended to live under robertjspencer.com.

The original site structure is still in place, but the user-facing page copy
has been rewritten so the site now presents as a personal website instead of
a stock demo.

Main pages:
- index.html: landing page
- generic.html: about page
- elements.html: supporting secondary page

Design base:
- Stellar by HTML5 UP
- html5up.net
